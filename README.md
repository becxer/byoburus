#byoburus

becxer's byobuable development environment

installation include tools below

 * dev set

    - htop
    - emacs
    - scikit
    - pydemon
    - java
    - iftop
    
 * vim set

    - vim
    - vim-plugin(jedi,NERDTree,tag,template)
    - ctags
    - vifm
    - .vimrc
    - .ctags
    
 * prompt set
 
    - color set prompt
    - useful alias
    - gitconfig & completion
    - git prompt

### HOW TO SETUP

 * To install dev set

        $./install-dev.sh
    
 * To install vim set
 
        $./install-vim.sh

 * To install prompt
 
        $./install-prompt.sh
